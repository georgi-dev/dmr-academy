Where do I start?

Volta brings over 40 premade pages with code examples and component variations. Each one of these pages uses the same structural markup.

The HTML

The structure of any webpage is the following:

<!DOCTYPE html>
<html lang="en">
  <head>
    ...
  </head>
  <body>
    ...
  </body>
</html>

Next, let's see what we can place inside of the <head> and <body> tags.

The Head

Next, let's see how the head markup of every Volta page looks like, including the one you're viewing. The Page Specific Links are directly related to additional functionality, so choose only the one you need for your page.

The Scripts

Next we need to add the scripts, the .js files. Scripts are used to add custom functionality to your page, such as form validation. Whenever you want to add a new script to your webpage, you use the script tag and point the path to the script file. Script tags always go at the bottom of the page, right before the <body> tag closes, as follows:

<!-- Theme -->
<script src="assets/js/application/themes/base.js" type="text/javascript"></script>
<script id="theme-script" src="assets/js/application/themes/volta.js" type="text/javascript"></script>

<!-- Document -->
